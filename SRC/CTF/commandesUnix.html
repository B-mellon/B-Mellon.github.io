Commandes Unix

Documentation of a command can be founded by typing
man commandName
grep "string" fileName
will search for every line containing the string in the given file
nc ipAdress port
connect to a host
nc -l portNumber
listen for connection on a selected port
ifconfig
lists a computer's network interfaces
host DNSname
gives the IP address for a given DNS name
ping target
helps to check if a remote computer is reachable from your computer, with target being a dns name or ip address
traceroute target
lists the routers that are located between my computer and the target (dns or ip)
cat /etc/services
list all ports and special assigned processes (names beside the port numbers are people who invented a certain protocol, usage of a certain port ...)
netsat -t
displays the active communication between your computer and remote ones
SSH enables secure communication channel between your computer and a remote one
ssh username@target
connect to a remote target (either LAN ip or other network ip)
exit
command to leave remote connection
Wireshark is an application that enables to look inside Internet packages
