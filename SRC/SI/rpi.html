RPI

L'hébergement du site est possible avec l'outil apache, qui va relier le port http au dossier /var/www/html de la rpi
sudo apt install apache2
installer apache
sudo service apache2 status
visualiser l'état du serveur
sudo service apache2 start
démarrer le serveur
sudo service apache2 stop
arréter le serveur
ifconfig
visualiser l'adresse IP locale du serveur
