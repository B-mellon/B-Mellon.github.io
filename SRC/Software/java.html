Java

Langage de programmation orienté objet et plateforme de dévellopement
Dévellopé en 1991 par Sun MicroSystems
Propriété depuis 2009 de Oracle

Avantage de Java : les programmes Java sont portables et peuvent être utilisé sans modifications
sous n'importe quelle plateforme (pas executé par l'ordinateur mais interprété par la JVM)

Java est composé de plusieurs blocs différents :

Le JDK
java development kit : utile aux programmeurs, il integre un ensemble d'outils: compilateur, executeur ... Il comprend le JRE, JVM et l'API java
LE JRE
java run time environment : permet d'exécuter du code java sur une machine
La JVM
java virtual machine : associé au JRE, c'est la machine virtuelle sur laquelle va s'éxecuter le code java
L'API Java
l'ensemble des entités et classes de base fournis par Java

Une fois le code java écrit, il doit d'abord être compilé, pour pouvoir ensuite être interprété par la JVM. Pour la compilation et l'exécution on peut se passe d'un IDE :
javac nomFichier.java
pour compiler le code .java et en créer un fichier .class executable par la JVM
java nomFichier
pour exécuter le code compilé
