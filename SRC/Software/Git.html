Quelques commandes utiles pour personnaliser son utilisation de Git

git config --global user.name “username”
renseigner son nom d'utilisateur, remplacer username par son nom d'utilisateur
git config --global user.email e-mail
renseigner son e-mail, remplacer e-mail par son e-mail
git config --global help.autocorrect
version 1 d'aide : proposer la commande la plus similaire
git config --global help.autocorrect nombre
version 2 d'aide : executer la commande la plus similaire au bout de nombre secondes
git config --global alias.abreviation commande
renseigner a git le choix d'un raccourcis pour une commande
git config --list
visualiser ses choix
git init
démarrer l'espace de travail git, à faire une fois pour chaque projet
git add *
indexer tout les fichiers pour la sauvegarde
git add *.type
indexer tout les fichiers d'un certain type
git add -i
indexer tout les fichiers de manière interactive
git commit -m ""
sauvegarder l'etat courant de tout les fichiers ayant ete ajoute grace a add
git commit -a -m ""
sauvegarder l'etat courant de tout les fichiers deja sous suivis
git commit --amend
apres un premier commit, commande pour le completer ou le renommer
git diff
visualiser les différences par rapport à la dernière sauvegarde

Visualiser l'historique des sauvegardes
git log
affiche l'historique des sauvegardes, si la liste est trop longue appuyer sur Entre pour poursuivre ou q pour quitter, peut prendre une liste de paramètres :
--graph
afficher l'historique sous la forme d'un arbre
-n
remplacer n par un nombre pour n'afficher que les n sauvegardes les plus récentes
--since=arg
n'afficher que les sauvegardes qui date de arg, arg au format Annee-Mois-Jour ou n.weeks, remplacer n par un nombre
-p
pour visualiser fichier par fichier les modifications apportés par chaque sauvegarde en comparaison à la précédente
--stat
Visualiser des statistiques des sauvegardes, nombre de fichiers modifiés ...

Cloner un projet hébergé sur serveur
git clone https://.../nomRepository
clone le projet se trouvant sur le serveur dans un nouveau dossier avec pour nom le nom du repository dans l'espace courant
git clone https://.../nomRepository nomDossier
meme commande que precedement mais avec choix du nom du dossier
Enregistrer le serveur et visualisation d'information
git remote -v
afficher les différents serveurs enregistrés, pour un clonage l'abréviation du serveur est origin
git remote add abreviation url
ajouter un serveur avec l'abreviation pour y acceder plus vite
git remote show abreviation
inspecter un serveur distant
git remote rename abreviationInitial abreviationSouhaite
renommer l'abreviation du serveur
git remote rm abreviation
supprimer un serveur
Communiquer avec le serveur
git fetch abreviation
recuperer le contenu du serveur, mais dans une autre branche crée pour l'occasion
git pull abreviation
récupérer et fusionner le serveur avec la branche locale du moment
git push abreviation branche
envoi vers le serveur sur la branche branche, le resultat de sa branche actuelle
